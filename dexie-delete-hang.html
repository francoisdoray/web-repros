Dexie Delete Hang

Demonstrating a hang when creating, opening, closing, and deleting databases in sequence.

This repro establishes a sequence: creating three separate databases, opening them, closing them, and finally deleting them. When run, this is expected to hang.

Status: Ready to start

What this script does

Sequence

// 1. Create 3 Dexie objects const db1 = new Dexie("HangDB1"); db1.version(1).stores({foo: 'id'}); // ... db2, db3 // 2. Open databases await Promise.all([db1.open(), db2.open(), db3.open()]); // 3. Close databases db1.close(); db2.close(); db3.close(); // 4. Delete databases await Promise.all([db1.delete(), db2.delete(), db3.delete()]);
